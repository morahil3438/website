What is the Purchase Data Management Portal?

This portal is a lightweight, client-side application designed to help you quickly record, track, and summarize your purchase transactions. It provides a simple, clean interface for managing individual purchase records and generating instant reports.

Key Features

  • Add Purchase: Easily input new records with item name, quantity, and price. Includes client-side validation for required and numeric fields.
  • View Records: See a detailed, tabular view of all stored purchases, including a unique ID for each transaction, and the calculated total price (Price x Quantity).
  • Reports: Instantly calculate key metrics, including the total number of purchases, the total amount spent across all records, and the highest value of a single purchase.
  • Data Management: Includes a **"Clear All Records"** button on the View page for bulk data deletion and a **"Reset"** button on the Add page for form cleanup.

Technology & Data Persistence

This application is built entirely using standard web technologies: **HTML**, **CSS**, and **JavaScript**. Crucially, all your purchase data is stored locally in your browser using **Web Storage (localStorage)**.

**Note on Local Storage:** This means your data is persistent even if you close the browser, but it is tied to the specific browser and device you are using. To function correctly, this project should be run using a local web server (e.g., accessed via http:// or https://, not file://).
